# 変更履歴

## v1.1.1 (2026-07-09)

### 対象範囲
- 「納品データ生成（雇用保険・離職票）」タブを対象に機能修正。
- 「公文書ビューア」タブの処理は変更なし。
- 共通ヘッダーのバージョン表示を `v1.1.1 (2026-07-09)` に更新。

### PDF自動分類の変更
- `お知らせ（離職された皆様へのダウンロード先ＨＰ）.pdf` を出力不要に分類。
- `厚労省からのお知らせ（20210501）.pdf` を出力不要に分類。
- 数字プレフィックス付きのファイル名にも対応。
- `お知らせ` / `おしらせ`、`ＨＰ` / `HP` 等の表記ゆれにも対応。

### 追加PDFの自動追加
- `離職されたみなさまへ（一般）.pdf` をHTML内に埋め込み。
- `正しく申告し、正しく受給しましょう！.pdf` をHTML内に埋め込み。
- 上記2件をデフォルトで従業員用に配置。
- 通常PDFカードと同じく、プレビュー、PDF保存、ファイル名編集、ドラッグ移動、ZIP出力に対応。

### 表示改善
- HTML内に埋め込まれて自動追加されるPDFに `ZIP外から自動追加` タグを表示。

### ZIP出力改善
- 出力対象ファイルがない空フォルダは、納品ZIPに作成しないよう変更。

### 不具合修正
- 結合PDFがフォルダ上部の件数表示に反映されない不具合を修正。
